CREATING A HELLO WORLD PROGRAM

After having downloaded and installed Nova, enter the following code into a file named HelloWorld.nova

class HelloWorld { public static main() { Console.writeLine("Hello, world!") } }

Then from command line, or terminal, navigate to the directory that you created the file in and run the following command:

novac HelloWorld.nova -o HelloWorld

If you see an error that looks something like: "'novac' is not recognized as an internal or external command, operable program or batch file.", follow the instructions explained here.

You should see a new executable file outputted in the folder that you ran novac in.

From the same command prompt window, you can run the HelloWorld program:

./HelloWorld

And you should see:

Hello, world!
